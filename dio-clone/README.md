# DIO Clone

Projeto de clone do site da [DIO](https://www.dio.me/), utilizando ReactJS.

### Esse projeto foi feito para praticar conhecimentos com:
- ReactJS
- Styled Components
- React Router Dom
- React Icons
- React Hook Form
- YUP
- Axios

### Foram clonadas as seguintes páginas:
- Home
- Login
- Cadastro
- Feed

### Foram criados os seguintes componentes:
- Header
- Button
- Input
- Card
- Likes
- UserInfo
- UserPicture

Para mim, foi um ótimo projeto, muito importante para pôr em prática e fixar os conhecimentos adquiridos. Vejo que ainda há muito a ser feito e tenho muito a aprender, mas esse foi um ótimo começo!
